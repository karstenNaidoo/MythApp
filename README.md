Overview

This is a native Android application developed using Kotlin in Android Studio. The app helps users determine whether common life hacks are real or urban myths through a flashcard-style quiz.

Features

* Welcome screen with start button
* Flashcard quiz with Hack or Myth options
* Feedback after each answer
* Next button to move through questions
* Score screen displaying total correct answers
* Performance message based on score
* Multiple screen navigation using intents

How It Works

1. User taps Start Quiz
2. A life hack statement is displayed
3. User selects Hack or Myth
4. Feedback is shown
5. User moves to next question
6. Final score is displayed on score screen

Technologies Used

* Kotlin
* Android Studio
* XML Layouts
* Android Activities and Intents
